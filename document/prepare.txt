our enviornment in windows :  link static and use shared runtime.

depend libs : antlr,llvm,boost,boost.log

boost & boost.log
1. download boost and boost.log(andysem version: http://sourceforge.net/projects/boost-log/ )
2. see http://stackoverflow.com/questions/13285714/boost-log-installation-error-on-boost-1-52-0 to patch boost.log
2. build option : {{{
     b2 variant=release link=static runtime-link=shared address-model=64
   }}}
   
antlr & antlr C target:
1. download antlr and antlr source from http://www.antlr3.org/download.html.
2. see https://github.com/antlr/antlr/issues/4 to remove antlr c target file.
3. compile c target.

prepare v8:
1. checkout from http://v8.googlecode.com/svn/trunk to c:\depends\v8
2. cd c:\depends\v8
3. run command : 
   svn co -q http://gyp.googlecode.com/svn/trunk build/gyp
   svn co -q http://src.chromium.org/svn/trunk/deps/third_party/cygwin@66844 third_party/cygwin
   svn co -q http://src.chromium.org/svn/trunk/tools/third_party/python_26@89111 third_party/python_26
4. then run command to generate sln files:
   third_party\python_26\python.exe build\gyp_v8 -Dcomponent=shared_library -Dtarget_arch=x64 -G msvs_version=2008
